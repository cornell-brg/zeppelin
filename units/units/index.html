Microarchitectural Units

Zeppelin’s pipeline is composed of the following first-class microarchitectural units, each connected to its neighbors through the standardized interfaces described in Data Flow in the Processor:

  • Fetch Unit (FU)
  • Decode Issue Unit (DIU)
  • Execute Units (XU)
  • Writeback Commit Unit (WCU)
  • Control Flow Unit (CFU)

Each unit is implemented as a single parameterizable SystemVerilog module under the corresponding hw/<name>/ directory. The toplevel hw/top/Zeppelin.v is what wires them together; see Processor Overview for the toplevel composition and the most important configuration parameters.
